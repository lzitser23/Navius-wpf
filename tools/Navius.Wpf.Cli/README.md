# navius-wpf (CLI)

Copy-paste component registry tool for
[navius-wpf](https://github.com/lzitser23/Navius-wpf): `navius-wpf add <item>` copies
a component's source, its full transitive dependency closure, and its theme
dictionaries into your project under your own namespace. `navius-wpf list` shows all
78 registry items.

Documentation: https://wpf.naviusui.dev/guides/install/
