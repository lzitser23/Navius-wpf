# Navius.Wpf.Ui

The styled layer of the [navius-wpf](https://github.com/lzitser23/Navius-wpf) suite:
token-themed `ControlTemplate`s and composite components (Sidebar, CommandPalette,
Card, Timeline, and more) over `Navius.Wpf.Primitives`, following the same
warm-grayscale token system as the web catalog and re-theming live with it.

Machine-readable contracts for every component: https://wpf.naviusui.dev
